Project Overview

Alphacons needed a digital registration system for its growing tech conference. I developed a full-stack platform that allowed attendees to register, pay via Stripe, and receive confirmations automatically. It also included an admin dashboard for non-technical staff to manage the event.

Key Features

  • Modern React frontend with intuitive registration UX
  • Node.js backend with MySQL database hosted on Google Cloud
  • Stripe integration for secure payments and invoicing
  • Admin dashboard for managing speakers, sessions, and users
