Social Media Sentiment Analysis

Python • Machine Learning • TensorFlow • MongoDB

Project Overview

This machine learning project analyzes social media posts to determine sentiment using natural language processing (NLP). Built in Python and powered by TensorFlow, it collects, classifies, and visualizes public sentiment in real-time.

Technical Stack

  • Used Python’s NLTK and SpaCy for data preprocessing and tokenization
  • Built and trained a neural network using TensorFlow and Keras
  • Stored scraped posts and prediction results in MongoDB
  • Achieved over 85% classification accuracy on validation data
  • Visualized sentiment trends using Matplotlib and Seaborn

Use Case

This tool is useful for analyzing audience sentiment during marketing campaigns, product launches, or news events. It was tested using data scraped from Reddit and X (formerly Twitter).

Built for learning, but with real-world impact. This project helped strengthen my NLP, Python scripting, and data pipeline design skills.
