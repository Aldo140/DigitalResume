Personal Finance Tracker App

  • Live Demo
  • Source code

The Personal Finance Tracker App is designed to help students manage their finances, focusing on student loan debt and repayment strategies. It provides features such as debt tracking, payment alerts, and financial education.

This project aims to educate students on minimizing high-interest debt while offering a clear picture of their financial standing.

  • Developed using React for a dynamic user experience.
  • Implemented financial data visualization features for better understanding.
  • Created a user-friendly interface tailored for students.
